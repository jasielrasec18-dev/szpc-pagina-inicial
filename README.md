📄 NeonCode — Landing Page

NeonCode é uma landing page simples e moderna criada com HTML e CSS, projetada para apresentar de forma clara um curso ou produto de Front-end. A página destaca benefícios, convida o visitante a começar e é ideal para uso como portfólio ou como template para educação e marketing.

🚀 Visão Geral

Este projeto demonstra uma página fixa (one-page), com navegação suave para seções como Início, Benefícios e Contato. O conteúdo é focado em:

📌 Aprendizado prático de Front-end

🧹 Código limpo e organizado

💼 Preparação para o mercado e empregabilidade

A página é responsiva e pode ser customizada para diferentes propósitos educativos ou comerciais.

🔗 Página ao vivo: https://jasielrasec18-dev.github.io/szpc-pagina-inicial/

🛠️ Tecnologias Utilizadas

Este projeto utiliza apenas tecnologias web básicas:

HTML – estrutura da página

CSS – estilos visuais e layout

Não há dependências externas ou frameworks — você pode abrir diretamente o index.html no navegador para visualizar.

👣 Instruções
🧾 Como visualizar

Você pode abrir o projeto de duas formas:

Diretamente no navegador:
Abra o arquivo index.html na raiz do projeto com um navegador moderno.

Via GitHub Pages:
A página já está publicada automaticamente através do GitHub Pages no link acima.

🧩 Estrutura do Projeto
📦 raiz do projeto
 ┣ 📜 index.html        # página principal
 ┣ 📜 styles.css        # estilos CSS
 ┗ 📁 /assets           # imagens ou recursos extras (opcional)
✨ Como personalizar

Você pode editar os textos, cores e conteúdos modificando o HTML e CSS. Algumas sugestões de personalização:

Alterar o título e as seções para refletir outro curso ou produto

Adicionar mais seções como Depoimentos, Sobre ou FAQ

Incluir imagens ou ícones relevantes ao tema

🧪 Template para prototipar landing pages
